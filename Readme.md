YOU CAN ADD AUTH VIA AN NGINX PROXY SERVER


YOU NEED TO HAVE DOCKER (COMPOSE) INSTALLED

IT WILL RUN AT `http://yourpc:8080`

IF YOU USE DOCKER on localhost it will be `http://localhost:8080`

---

Run:

docker-compose up
